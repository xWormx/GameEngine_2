1) 	Jag har använt den rekommenderade utvecklingsmiljön, så kan byggas m.h.a 'Makefile'. 
	För att göra detta i VS Code så göra man "Add Folder to Workspace" på mappen GameEngine_2.

2)	- När spelet startar uppstår en meny med knappar som man kan trycka på m.h.a musklick.
	
	- För att starta spelet trycker man på "Start Game".

	- Välj namn för spelare 1 och 2 genom att klicka på respektive textfält med musklick, då aktiveras
	  textfältet och du kan skriva vilket namn spelaren ska ha. Tryck sedan på "Enter"-knappen för att 
	  bekräfta namnet.

	- Tryck sedan på "Start Game" för att starta själva spelet.

	- I spelet styrs figurerna med tangentbordet:
		
		Spelare 1: (w,a,s,d) 		skjuter på (b)
		Spelare 2: (piltangenterna) 	skjuter på (p)
	
	- Om en robot släpps förbi när tiden gott ut och den sista roboten har dödats eller gått förbi förlorar ni.
	- Om alla robotar skjuts ihjäl vinner ni.
	
	- När spelet är slut visas en slutscene med resultatet och en "Quit Game"-knapp som går att trycka på med musklick.
